\# Tyche — Claude Code Context



\## Project

On-chain prediction market reputation protocol on Somnia blockchain.

Tagline: "Your on-chain prediction identity."

Hackathon: Somnia Reactivity Mini Hackathon 2026



\## Tech Stack

\- Smart Contracts: Solidity + Hardhat + Somnia Reactivity SDK

\- Backend: Node.js + Fastify + PostgreSQL + Redis

\- Frontend: Next.js 14 + Tailwind + wagmi + viem + RainbowKit

\- Indexer: Custom event listener subscribed to Somnia RPC
